# Privacy Policy — Radio Ceylon

**Last updated:** 2026-03-29

## Data Collection

Radio Ceylon does not collect, store, or transmit any personal data.

## Local Storage

The extension stores the following data locally on your device using Chrome's storage API:

- **Theme preference** (dark or light mode)
- **Last played station** (station identifier)
- **Cached station list** (station metadata, refreshed hourly)

This data never leaves your device and is not accessible to the extension developer or any third party.

## Network Requests

The extension makes network requests only to:

1. **GitHub Pages** — to fetch the list of available radio stations (public JSON file)
2. **Radio stream servers** — to play the selected radio station's audio stream

No analytics, tracking, or telemetry of any kind is included.

## Contact

For questions about this privacy policy, open an issue on the project's GitHub repository.
